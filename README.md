# バチェラー式恋愛診断
ミニアプリweek2025に提出するため、即席で完成させたアプリです。
HTML/CSS/JSのみで作成しています。
筆者がバチェラーのファンであり、そこで得た偏った恋愛のノウハウを詰め込みました。
あなたは真実の愛を得られるのでしょうか・・・

# サービスURL
https://bachelor-diagnosis.onrender.com/

# 使い方
startボタンを押し5つの質問に答えてください。最後、結果が表示されます。
結果はTrue endとBad endのみです。
True endの方は恋愛上級者です。バチェラーバチェロレッテにぜひご参加ください。
最初に戻るボタンで最初から診断をやり直すことができます。
またXに投稿しシェアすることができます。

# 工夫した点
背景のアニメーションをつけてユーザーにリッチな体験ができるようにしました。
