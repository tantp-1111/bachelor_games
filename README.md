# バチェラー式恋愛診断
![app image](images/ogp_image.png)

## 概要
ミニアプリweek2025に提出するため、即席で完成させたアプリです。<br>
筆者がバチェラーのファンであり、そこで得た偏った恋愛のノウハウを詰め込みました。<br>
 あなたは真実の愛を得られるのでしょうか・・・

## サービスURL
https://bachelor-diagnosis.onrender.com/

## 技術スタック

HTML/CSS/javascript

## 使い方
startボタンを押し5つの質問に答えてください。最後、結果が表示されます。
結果はTrue endとBad endのみです。
True endの方は恋愛上級者です。バチェラーバチェロレッテにぜひご参加ください。
最初に戻るボタンで最初から診断をやり直すことができます。
またXに投稿しシェアすることができます。

## 工夫した点
画面背景にパーティクルアニメーションを実装しました。ゆらゆらと漂う光の演出によって、バチェラーのロマンティックで華やかな夜の雰囲気を表現しました。シンプルな診断アプリでありながらユーザーが番組の世界観に没入できるよう意識し、印象に残るUIを目指しました。
